# Active Directory Home Lab

## Overview
This project documents the process of building a virtualized lab environment to simulate an enterprise network using Windows Server, Linux, and security tools.

## Lab Environment
- Windows Server 2022 (Domain Controller)
- Ubuntu Linux
- Kali Linux

## Objectives
- Deploy and configure Active Directory Domain Services (AD DS)
- Create and manage users and groups
- Configure virtual networking between systems
- Perform basic troubleshooting and connectivity testing
- Introduce basic security testing using Kali Linux
